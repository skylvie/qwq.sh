# [qwq.sh](https://qwq.sh)
```
index.html - main page
dmca.html - dmca info page
404.html - 404 error page
src/ - assets
    > style.css - CSS for index.html and 404.html
    > index.js - JS for index.html
    > 404.js - JS for 404.html
sh/ - shell scripts
    > gitow.sh - install gitow
    > roku.sh - install rokucli
cfg/ - config file(s)
    > Caddyfile - caddy config
ascii/ - ASCII art
    > *.txt - ASCII arts
    > manifest_small.json - ASCII art on index.html
    > manifest_big.json - ASCII art on 404.html
```
